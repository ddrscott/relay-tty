---
title: Voice Input
---


Dictate commands to the terminal using your device's speech recognition.

## Using voice input

1. Tap the **microphone** icon in the mobile toolbar
2. Speak your command
3. The transcribed text appears in the scratchpad
4. Review and send to the terminal

Voice input uses the Web Speech API built into your browser. It works best on:

- Chrome (Android and desktop)
- Safari (iOS and macOS)

<Callout type="info">
Voice input transcribes to the scratchpad, not directly to the terminal. This lets you review and edit before sending.
</Callout>
